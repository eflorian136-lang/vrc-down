# VRC VOD Downloader 🚀

Ein einfaches Tool, um Twitch-VODs direkt über das Web herunterzuladen.

## 🛠️ Verwendung
1. Öffne die Webseite: https://eflorian136.github.io/vrc-down/
2. Gib die Twitch-URL deines VODs in das Feld ein.
3. Klicke auf **Download**.
4. Dein Video wird im Hintergrund verarbeitet.

## 🔒 Sicherheit
Dieses Projekt ist Open-Source. Deine Daten werden nicht gespeichert. Das System nutzt GitHub Actions für eine sichere und automatisierte Verarbeitung.

## 💡 Mitmachen
Da dieses Projekt öffentlich ist, kannst du gerne Verbesserungen vorschlagen oder Fehler melden.
